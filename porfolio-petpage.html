For instance, if you need an intro to your page, make the first box a content box such as this one. Note that the tallleft class should only go in the leftmost space of the grid, while the tallright class should only go in the rightmost space.

To use the cropped image feature, you'll need the code style="margin-top: -400px; margin-left: -50px; width:1024px; in your img tag, like I've done with the first image on the page. Simply replace any or all of the numbers until the image is cropped how you want it. Viewers can still see the full version by right clicking and choosing view image, or by dragging the image to the address bar.
